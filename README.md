# TukangAbsen 
An Android app written in Kotlin to hack your attendance steps becomes easier and faster.

### Libraries Used
- Retrofit2
- Gson
- Picasso
- Encryption
- Okhttp Interceptor
- RxJava (next)
- Dagger2 (next)
